Hospital Emergency Room (ER) Performance Dashboard

Description

The dashboard presented in this Power BI report provides a thorough analysis of the operations of a hospital's Emergency Room, including patient flow, waiting times, admission rates, and levels of satisfaction. The report allows stakeholders to effectively monitor the ER, identify any issues, and make informed decisions to enhance the patient experience.

Key Technologies Used 

Power BI Desktop (.pbit template)

Power Query – Data cleaning and transformation

DAX (Data Analysis Expressions) – Calculated columns and measures

Data Modeling – Relationship building and data hierarchies

Interactive Visualizations – KPI cards, bar charts, line charts, slicers

Data

The dashboard is built using a structured Hospital Emergency Room dataset containing patient-level information from Kaggle, including

Patient ID and demographic details (Age, Gender, Race)

Admission date and time

Department referrals

Admission status

Patient wait time

Patient satisfaction scores
